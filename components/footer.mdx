# Footer Component

The **Footer** component provides site-wide navigation, branding, social media links, and legal information. Features a clean multi-column layout with gradient accent line.

## Features

- **Multi-column layout**: Brand, plans, company, and contact sections
- **Social media integration**: Facebook, Instagram, Twitter/X icons
- **Dynamic year**: Auto-updates copyright year
- **Organized link groups**: Categorized navigation
- **Legal links**: Terms and privacy policy
- **Responsive design**: Stacks columns on mobile
- **Gradient accent**: Decorative top border

## Dependencies

No external dependencies - uses vanilla React:

```jsx
import './Footer.css'
```

## Dynamic Copyright Year

The component automatically displays the current year:

```jsx
const year = new Date().getFullYear()

// Later in JSX:
<p className="footer__copy">© {year} Telenet Fibra Óptica. Todos los derechos reservados.</p>
```

## Customization

### Update Social Links

```jsx
<a href="https://facebook.com/yourpage" className="footer__social" aria-label="Facebook">
    {/* SVG */}
</a>
<a href="https://instagram.com/yourprofile" className="footer__social" aria-label="Instagram">
    {/* SVG */}
</a>
<a href="https://twitter.com/yourhandle" className="footer__social" aria-label="Twitter">
    {/* SVG */}
</a>
```

### Modify Navigation Links

Edit the link arrays in each column:

```jsx
<div className="footer__col">
    <h4 className="footer__col-title">Products</h4>
    <ul className="footer__links">
        <li><a href="/product1">Product 1</a></li>
        <li><a href="/product2">Product 2</a></li>
    </ul>
</div>
```

### Change Contact Information

```jsx
<li><a href="https://wa.me/+1234567890">WhatsApp</a></li>
<li><a href="mailto:contact@yourcompany.com">Email</a></li>
<li><a href="tel:+1234567890">Phone: (123) 456-7890</a></li>
```

### Update Branding

```jsx
<div className="footer__logo">
    <img src="/your-logo.png" alt="Your Company Logo" className="footer__logo-img" />
    <span className="footer__brand-name">Your<span>Company</span></span>
</div>
<p className="footer__tagline">
    Your company tagline or mission statement goes here.
</p>
```

## Accessibility

- Semantic `<footer>` element
- `aria-label` attributes on icon-only links
- Descriptive alt text on logo image
- Proper heading hierarchy with `<h4>` column titles
- Focus states on links (handled by CSS)
