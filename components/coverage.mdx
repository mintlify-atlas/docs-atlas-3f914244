# Coverage Component

The **Coverage** component displays Telenet's headquarters location using a Google Maps embed within a modern bento grid layout. Features animated cards with office information, hours, and directions.

## Features

- **Google Maps embed**: Interactive map showing office location
- **Bento grid layout**: Responsive card-based design
- **Office information**: Address, hours, and contact details
- **Staggered animations**: Cards animate in sequence
- **Direct navigation**: Click-to-open in Google Maps
- **Responsive design**: Adapts to mobile and desktop layouts

## Dependencies

```jsx
import { motion } from 'framer-motion'
```

- **Framer Motion**: For card entrance animations

## Google Maps Configuration

The component uses a Google Maps embed URL:

```jsx
const GOOGLE_MAPS_EMBED = "https://maps.google.com/maps?q=Corporacion+Telenet+de+Venezuela,+Barinas&t=&z=17&ie=UTF8&iwloc=&output=embed";
```

### Map URL Parameters

<ParamField path="q" type="string">
  Location query: "Corporacion Telenet de Venezuela, Barinas"
</ParamField>

<ParamField path="z" type="number">
  Zoom level: 17 (street-level view)
</ParamField>

<ParamField path="output" type="string">
  Set to "embed" for iframe embedding
</ParamField>

## Bento Grid Layout

The component uses a 12-column bento grid with 4 cards:

1. **Map Card** (8 columns): Interactive Google Maps iframe
2. **Address Card** (4 columns): Office location details
3. **Hours Card** (4 columns): Business hours
4. **Action Card** (4 columns): Link to open in Google Maps

## Animation Configuration

### Container Animation (Stagger Children)

```jsx
<motion.div
    className="bento-grid"
    initial="hidden"
    whileInView="visible"
    viewport={{ once: true, margin: "-100px" }}
    variants={{
        hidden: { opacity: 0 },
        visible: {
            opacity: 1,
            transition: {
                staggerChildren: 0.15
            }
        }
    }}
>
```

- Triggers when scrolled into view
- 150ms delay between each card

### Map Card Animation

```jsx
<motion.div
    className="bento-card bento-map"
    variants={{
        hidden: { opacity: 0, scale: 0.95 },
        visible: { opacity: 1, scale: 1, transition: { duration: 0.8, ease: "easeOut" } }
    }}
>
```

### Info Cards Animation

```jsx
<motion.div
    className="bento-card bento-info"
    variants={{
        hidden: { opacity: 0, x: 20 },
        visible: { opacity: 1, x: 0, transition: { duration: 0.6, ease: "easeOut" } }
    }}
>
```

- Slides in from the right (20px offset)
- Faster animation (600ms)

## Customization

### Change Location

```jsx
const GOOGLE_MAPS_EMBED = "https://maps.google.com/maps?q=Your+Business+Name,+City&t=&z=17&ie=UTF8&iwloc=&output=embed";
```

### Update Address

```jsx
<p>Your Street Address<br />
    City, State ZIP<br />
    Country</p>
```

### Modify Hours

```jsx
<div className="hours-content">
    <div className="hour-row">
        <strong>Mon – Fri</strong>
        <span>9:00 AM – 6:00 PM</span>
    </div>
    <div className="hour-row">
        <strong>Saturday</strong>
        <span>10:00 AM – 2:00 PM</span>
    </div>
    <div className="hour-row">
        <strong>Sunday</strong>
        <span>Closed</span>
    </div>
</div>
```

### Update Google Maps Link

Replace the full URL in the action card's `href` attribute with your location's Google Maps URL.

## Accessibility

- **iframe title**: Descriptive title for screen readers
- **loading="lazy"**: Deferred loading for performance
- **allowFullScreen**: Enables fullscreen mode in map
- **referrerPolicy**: Privacy protection
- **Semantic HTML**: Proper heading hierarchy
